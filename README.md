Добрый день, для вас доступны 2 профиля, Shadowsocks и VMess, на данный момент они стабильны. 

Для десктопа, мне нравится использовать приложение Outline с протоколом ss(shadowsocks) другие протоколы не умеет, его можно скачать по ссылке, внизу страницы https://getoutline.org/intl/ru/get-started/#step-3

Если используется иобильная сеть то лучше приложение Happ и протокол VMess (так же работает и с shadowsocks), скачать можно https://www.happ.su/main/ru
Если вам привычны другие приложения поддерживающие VMess, можно использовать их. Логика добавления ключей похожа у всех.

Оба приложения интуитивно понятны, в правом верхнем углу есть значок "+" и далее добавить из буфера обмена.

Напротив вашего имени есть 2 столбца, с подписью VMess и Outline/Shadowsocks. Первую ссылку нужно вставить в приложение Happ, вторую соответственно Outline.
